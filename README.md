# Computer Vision & Python Projects

This repository contains a collection of Python projects built while learning Artificial Intelligence, Computer Vision, and Automation.

## Projects Included

### Face Detection
Detects human faces in images using OpenCV Haar Cascade classifiers.

### Face & Eye Detection
Identifies faces and eyes in images using computer vision techniques.

### Person Counting
Counts the number of people in an image using YOLOv8 object detection.

### Helmet Detection
Detects whether a person is wearing a helmet using image processing and object detection techniques.

### Voice Assistant
A simple voice-controlled assistant built using Speech Recognition and Text-to-Speech libraries.

### Lyrics Finder
Fetches song lyrics using Python libraries and APIs.
